SvelteKit Agendas App

Dies ist eine einfache SvelteKit-Anwendung, mit der du deine Agendapunkte verwalten kannst. Du kannst neue Punkte hinzufügen und bestehende löschen. Das Projekt zeigt die grundlegenden Konzepte von Svelte, darunter Reaktivität und Event-Handling.

Die App basiert auf SvelteKit und nutzt eine reaktive Datenstruktur, um Änderungen in Echtzeit anzuzeigen. Neue Agendapunkte werden direkt in die Liste aufgenommen, und mit einem einfachen Klick können sie wieder entfernt werden. Die Benutzeroberfläche ist minimalistisch und intuitiv gestaltet, um eine einfache Nutzung zu ermöglichen.

Der Code verwendet moderne Web-Technologien wie TypeScript und CSS für eine ansprechende Gestaltung. Zudem ist die Anwendung leicht erweiterbar und kann für verschiedene Anwendungsfälle angepasst werden.

Installation & Start

Um die Anwendung lokal auszuführen, folge diesen Schritten:

# Repository klonen
git clone <repository-url>
cd my-agenda-app

# Abhängigkeiten installieren
npm install

# Entwicklungsserver starten
npm run dev

Nach dem Start ist die Anwendung unter http://localhost:5173 erreichbar.
